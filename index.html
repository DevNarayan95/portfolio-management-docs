ðŸ“‹ Core Requirements

Functional Requirements

1. Authentication Module

  • User Registration: Email validation, password strength validation, unique email constraint
  • User Login: JWT token generation, token refresh mechanism, secure password hashing
  • User Logout: Token invalidation and session management

2. Portfolio Management

  • Create multiple portfolios per user
  • Portfolio naming and description
  • Default portfolio creation on first login
  • Support for 4 investment types: Mutual Funds, Stocks, Bonds, Cryptocurrency

3. Investment Management

  • Add/Edit investments with purchase price and quantity
  • SIP (Systematic Investment Plan) management
  • Investment notes and historical tracking
  • Soft delete capability for data retention

4. Dashboard

  • Portfolio overview with total value and invested amount
  • Performance metrics and gain/loss calculations
  • Visual representations (pie charts, graphs, trends)
  • Asset-wise breakdown and ROI calculations

5. Transaction History

  • Complete buy/sell transaction logging
  • Filtering by date, type, and portfolio
  • Transaction details and audit trail

ðŸ”Œ API Design & Endpoints

Base URL

http://localhost:3000/api/v1

Authentication Endpoints

POST /auth/register

Register new user with email and password

POST /auth/login

Authenticate user and return JWT tokens

POST /auth/refresh-token

Refresh access token using refresh token

POST /auth/logout

Invalidate tokens and logout user

Portfolio Endpoints

GET /portfolios

Get all user portfolios

GET /portfolios/:id

Get portfolio by ID with investments

POST /portfolios

Create new portfolio

PUT /portfolios/:id

Update portfolio details

DELETE /portfolios/:id

Delete portfolio (soft delete)

Investment Endpoints

GET /portfolios/:id/investments

Get all investments in portfolio

POST /portfolios/:id/investments

Add new investment

PUT /portfolios/:id/investments/:invId

Update investment details

DELETE /portfolios/:id/investments/:invId

Delete investment

Transaction & Dashboard Endpoints

GET /portfolios/:id/transactions

Get transaction history with filtering

POST /portfolios/:id/investments/:invId/transactions

Add buy/sell transaction

GET /dashboard/summary

Get dashboard metrics and analytics
